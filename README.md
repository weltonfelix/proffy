## :notebook: Getting Started
1. Clone this repository
2. `cd server`
3. `yarn`
4. Setup your Postgres database.
5. Rename `.env.template` to `.env`.
6. Add your host url and database information.
7. `yarn knex:migrate`
8. `cd ../web`
9. `yarn`
10. `cd ../mobile`
11. `yarn`
12. `cd ..`
13. To start server
   1. `cd server`
   2. `yarn dev`
14. To start web
    1. `cd web`
    2. `yarn start`
15. To start mobile
    1. `cd mobile`
    2. `yarn start`
16. Enjoy!
